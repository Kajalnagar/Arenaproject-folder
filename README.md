## Program Description

This program is a simple Python application that interacts with the user. It asks the user a few basic questions such as their name, age, and favorite hobby. The answers provided by the user are stored in variables. After collecting the information, the program displays a friendly welcome message that includes the user’s details. The purpose of this program is to practice basic Python concepts like taking user input, storing data, and displaying output.

## Technical Requirements Implemented

This project fulfills the required technical conditions for the task:

- The input() function is used to ask the user questions and collect their responses.
- Variables are used to store the information entered by the user.
- The print() function is used to display the final welcome message.
- The program asks at least three questions to gather user information.
- The final output message is written in a friendly and easy-to-read format.

The project is organized in a simple structure to keep everything clear and easy to understand:

README.md – Contains the explanation and documentation of the project.
